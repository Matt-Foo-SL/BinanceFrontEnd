run command: npm run dev

1. Start backend server
2. Click on refresh in the browser until Backend connection is "Connected"
3. Click Get latency button, a number should appear below for example 123ms
4. Click Start stream button, the graph should populate
5. Click Stop to stop stream, after a few seconds status should change to "Stopped"
6. Click Simulate stream crash and auto-reconnect button, the graph should stop and start again after a few seconds
